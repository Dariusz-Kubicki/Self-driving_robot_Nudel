# Self-driving_robot_Nudel
Dokumentacja mojego pierwszego robota o nazwie "Nudel". Robot jest prosty i składa się z zestawu do budowy robota z Aliexpress, napędzany jest 4 silnikami DC z przekładniami TT, początkowo robot ma jeździć za czarną linią (line follower) ale w przyszłości planuje dodać do niego czujniki odległości lub nawet kamerę.
